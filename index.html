About Raw Data Media Player

Version:

Raw Data Media Player is an advanced web-based tool for visualizing and playing binary data as both video and audio. It supports multiple color modes, PCM formats, and provides real-time visualization features.

Built with:

  • HTML5 Canvas for video rendering
  • Web Audio API for audio playback
  • CSS3 for styling and animations

Features:

  • Multiple color modes (RGB32, RGB24, RGB565, etc.)
  • Various PCM audio formats (8-bit to 32-bit, including float)
  • Real-time waveform and XY oscilloscope
  • Custom playback speeds
  • Screenshot capability
  • Canvas size adjustment
  • Volume control
  • File conversion tools
